1. Задача прогнозирование цены на такси в Нью-Йорке
Данные:
 -vendor ID (идентификатор продавца)
 -rate code (код ставки)
 -passenger count (количество пассажиров)
 -trip time (время поездки)
 -trip distance (расстояние поездки) 
 -payment type (способ оплаты)
2. Общей задачей регрессии является прогнозирование некоторого непрерывного значения для заданных параметров.
Например:
 -Прогноз стоимости дома на основе количества комнат, местоположения, года постройки и т. д.
 -Прогноз расхода топлива автомобиля на основе типа топлива и параметров автомобиля
 -Прогноз оценки времени для устранения проблемы на основе атрибутов проблемы.
  Общей особенностью всех этих примеров является то, что параметр, который мы хотим предсказать,
может принимать любое числовое значение в определенном диапазоне. Другими словами, это значение представляется
целочисленным или float / double, а не enum или логическим типом.
3. Решение

Строим модель ML -> Обучаем модель на основе существующих данных -> Оцениваем, насколько она хороша -> Прогнозируем тариф на такси
